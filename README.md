## 🛠️ ECOSISTEMA DE SOFTWARE / SOFTWARE ECOSYSTEM

La suite **BLOCKpro** es una plataforma bilingüe diseñada para la gestión profesional de proyectos de construcción y logística.

* **EstimaPRO:** Motor de cálculo técnico para análisis de insumos, costos y presupuestos.
* **FamPro:** Módulo de gestión y control técnico de proyectos.
